Project 3 - Optimization-Based Animation

Write Up:

Author - Levi Cavagnetto

EXTERNAL LIBRARIES:

No external libraries were used to make this project. Most of the project source code was modified from the in-class activities.


DIFFICULTIES:

1. Obstacles were very difficult to implement with my implementation of the IK. I only had control over the joints through the endpoint, and could not find a way to change their position.


FEATURES:

Part 1:
- Multi-arm IK (2D). I was able to create a multi-arm IK.
- Joint limits. I added joint limits to my IK skeleton.
- Obstacles. Obstacles were difficult to figure out, but the end effector of the IK cannot go through the floor.
- User Interaction. The user can control both arms using left and right click respectively. And 'a' and 'd' move the root of the IK left and right.
- Moving IK. The user can move the root of the IK to the left and right using 'a' and 'd'.
- 3D rendering. The IK is not simulated in a 3D environment but the IK is rendered in 3D.

Part 2:
- For Part 2, I implemented the FABRIK solver. See the demonstration below.
