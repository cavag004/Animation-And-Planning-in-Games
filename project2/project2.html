Project 2 - Physically-Based Animation with PDEs

Write Up:

Author - Levi Cavagnetto

EXTERNAL LIBRARIES:

No external libraries were used to make this project. Most of the project source code was modified from the in-class activities.


DIFFICULTIES:

1. Wind was difficult to get working for Part 1. Everything else went fairly smooth.


FEATURES:

Part 1:
- Multiple Ropes (2D). I was able to add multiple ropes into my simulation.
- Cloth Simulation. I was able to connect the ropes horizontally.
- High-quality Rendering. I added a controllable camera and colored the cloth.
- Air Drag for Cloth. I was able to add drag to the cloth simulation.
- User Interaction. The user can control the obstacle and move it through the cloth to tear it.
- Ripping/Tearing. The user can tear the cloth with the obstacle or by clicking different spots on the cloth to delete nodes.

Part 2:
- For Part 2, I implemented the Shallow Water equation. See the demonstration below.
